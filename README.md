# Chrome DevTools MCP Bridge 🧪⚙️

MCP server that exposes Chrome DevTools Protocol as MCP tools for AI agents.

## Why

Debugging AI agents that interact with browsers is hard. This bridge gives AI agents direct access to browser debugging capabilities via the MCP protocol.

## Features

- **chrome_navigate**: Navigate to URLs
- **chrome_screenshot**: Capture page screenshots  
- **chrome_evaluate**: Execute JavaScript in page context
- **chrome_dom_snapshot**: Get DOM structure
- **chrome_network_logs**: Capture network requests

## Quick Start

```bash
# Install
npm install
npm run build

# Run with Claude Desktop or other MCP clients
npm start
```

## MCP Tool Schema

Add to your MCP client config (e.g., Claude Desktop):

```json
{
  "mcpServers": {
    "chrome-devtools": {
      "command": "node",
      "args": ["/path/to/chrome-devtools-mcp-bridge/dist/index.js"],
      "env": {
        "CHROME_PATH": "/usr/bin/google-chrome"
      }
    }
  }
}
```

### Tools

| Tool | Input | Description |
|------|-------|-------------|
| `chrome_navigate` | `{ url: string }` | Navigate to URL, wait for networkidle0 |
| `chrome_screenshot` | `{ fullPage?: boolean }` | Screenshot (default viewport or full page) |
| `chrome_evaluate` | `{ script: string }` | Execute JS in page context |
| `chrome_dom_snapshot` | `{ maxLength?: number }` | DOM innerHTML (default 10k chars) |
| `chrome_network_logs` | `{}` | Capture Network.requestWillBeSent events |
